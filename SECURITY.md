# Security Policy

## Supported Versions

| Version | Supported |
| ------- | ------------------ |
| 1.x.x | :white_check_mark: |
| < 1.0 | :x: |

## Reporting a Vulnerability

**Do not** open a public issue for security vulnerabilities.

Instead, please report via:

- **GitHub Security Advisories**: [Report a vulnerability](https://github.com/Hyggshi-OS-project-center/NexCode/security/advisories/new)

### Information to Provide

- Description of the vulnerability and potential impact
- Reproduction steps
- Affected NexCode version
- Operating system/environment

### Response Time

- **Report Receipt Confirmation**: within 3 business days
- **Preliminary Assessment**: within 7 days
- **Patch/Result Notification**: depending on severity, usually within 30 days

### Scope

Issues covered by reporting:

- Remote Code Execution (RCE) related to Monaco Editor extensions
- Vulnerabilities in loading/executing files from workspace
- Privilege escalation In Electron shell

Outside scope:

- Common UI/UX errors

- Features not yet implemented

Thanks for making NexCode safer!
